# Zaykaa Start - Global project requirements
#
# This is the single root-level requirements reference for the full project.
# The project is configured to use your system Python installation, not a venv.
# It includes:
# - system prerequisites
# - environment requirements
# - frontend package references
# - backend Python packages that can be installed with pip
#
# System prerequisites
# - Python 3.13+ (tested in this workspace with Python 3.13)
# - Node.js 22.14.0 tested
# - npm 10.9.2 tested
# - MySQL 8.x running locally
#
# Required databases
# - zaykaa_db for the legacy compatibility backend
# - zaykaa_user_service for the new user microservice
# - zaykaa_chef_service for the chef microservice
# - zaykaa_booking_service for the booking microservice
# - zaykaa_order_service for the order microservice
# - zaykaa_payment_service for the payment microservice
#
# Current backend environment values used by the app
# - FLASK_ENV=development
# - JWT_SECRET=replace_with_a_long_random_jwt_secret
# - DB_USER=root
# - DB_PASSWORD=your_mysql_password
# - DB_HOST=localhost
# - DB_NAME=zaykaa_db, zaykaa_user_service, zaykaa_chef_service, zaykaa_booking_service, zaykaa_order_service, or zaykaa_payment_service depending on the service
#
# Current frontend environment value used by the app
# - REACT_APP_API_URL=http://localhost:5000/api
#
# System-wide setup example
# 1. Install backend packages into your local system Python:
#    python -m pip install -r requirements.txt
#    If `python` is not on PATH in Windows, use:
#    py -3 -m pip install -r requirements.txt
# 2. Install frontend packages:
#    npm install --prefix zaykaa-frontend
# 3. Start frontend, gateway, user service, chef service, booking service, order service, payment service, and legacy backend together from the repo root:
#    npm start
#
